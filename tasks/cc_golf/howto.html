Welcome!

Thanks for playing our SpreadIT Code Golf!
The task is simple: write the program solving the task described below and try to make it as small as possible. Any byte counts! The system is counting spaces and new lines as wel ;) On the main page you can insert your code and it will be run and tested on backend. You will receive feedback (error or success info) and you can submit multiple times to improve your score (it will be overwritten based on your email).

There are 5 languages to choose from: Python 3.6, Ruby 2.4, bash 4.4, PHP 7.1 and Node.js 8.9

After successful attempt you will receive a random nickname, which we will call you by when announcing winners. You have to be present at the announcement at Clearcode stand at 14:45 today to get the prize (try to remember you nickname, but we can verify with email as well). If you won't be present, the next person in leaderboards will get the prize.

Huge thanks to firemark of Coding Dojo Silesia for letting us use the code golf platform <3

Do you need more help? Maybe something is not working? Visit us at Clearcode stand on the conference and we will help you any way we can ;)
